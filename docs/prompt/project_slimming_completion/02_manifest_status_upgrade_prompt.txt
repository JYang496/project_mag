1. 复查是否还有未登记 scene：

```powershell
rg --files tests/scenes -g "*.tscn"
```

2. 复查是否有 runner-only script 需要补 scene wrapper：

```powershell
rg --files tests/headless -g "*.gd"
```

3. 运行全 manifest Worker：

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File tests/infrastructure/run_test_workers.ps1 -GodotPath 'D:\SteamLibrary\steamapps\common\Godot Engine\godot.windows.opt.tools.64.exe' -Jobs 2 -OutputRoot 'test-results\manifest-full-candidate'
```
